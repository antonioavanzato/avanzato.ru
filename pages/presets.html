Частые вопросы

Пресеты — это набор настроек для обработки фото (Lightroom, Camera RAW, Capture One), которые применяются прямо внутри программы.

LUT'ы — это таблицы преобразования цвета для видео, которые используются в DaVinci Resolve, Premiere Pro и других видеоредакторах.

Проще говоря: пресеты — для фото, LUT'ы — для видео.
Пресеты лучше всего работают на RAW-файлах и при нормальной экспозиции. Это не «волшебная кнопка», а инструмент ускорения работы.

Иногда достаточно просто подправить экспозицию, баланс белого или контраст — и результат будет идеальным.
Нет. Все пресеты и LUT'ы рассчитаны на простое и быстрое применение, даже если вы только начинаете. При этом они подходят и для профессионалов, которым важно сохранить единый стиль и ускорить обработку.
Да. Вы можете использовать их для личных и коммерческих съёмок: клиентов, соцсетей, рекламы, YouTube и т. д.

Единственное ограничение — перепродажа или передача файлов третьим лицам запрещена.
Цвет может немного отличаться из-за экранов и настроек устройств. Пресеты и LUT'ы создаются с учётом стандартных цветовых пространств, но финальный результат всегда лучше проверять на нескольких экранах.

LR Lightroom Classic / CC (macOS / Windows)

1. Разархивируй скачанный ZIP-файл. Внутри будет папка Lightroom Presets с файлами формата .xmp

2. Открой Lightroom, перейди в модуль Develop (Коррекция). В левой панели найди раздел Presets

3. Нажми + рядом с заголовком Presets → выбери Import Presets → укажи все .xmp файлы из папки

4. Готово! Пресеты появятся в разделе User Presets. Нажми на любой, чтобы применить к фото

💡 Альтернатива: скопируй .xmp вручную в:
macOS: ~/Library/Application Support/Adobe/Lightroom/Develop PresetsWindows: C:\Users\[Имя]\AppData\Roaming\Adobe\Lightroom\Develop Presets

Mobile Lightroom Mobile (iOS / Android)

1. Перенеси .dng файлы на телефон через AirDrop, Telegram, Google Drive или iCloud в приложение «Файлы»

2. В Lightroom Mobile нажми + → Добавить фото → выбери .dng файл. Он откроется с уже применёнными настройками

3. Нажми ⋮ (три точки) в правом верхнем углу → Создать пресет → дай название и сохрани

4. Теперь открывай любое своё фото → панель Presets → выбирай сохранённый пресет

⚠️ Важно: мобильные пресеты — это .dng файлы, а не .xmp. Каждый .dng — «носитель» настроек. После создания пресета исходный .dng можно удалить.

ACR Adobe Camera Raw (Photoshop / Bridge)

1. Скопируй .xmp файлы в папку:
macOS: ~/Library/Application Support/Adobe/CameraRaw/SettingsWindows: C:\Users\[Имя]\AppData\Roaming\Adobe\CameraRaw\Settings

2. Перезапусти Photoshop или Bridge. Пресеты появятся в панели Camera Raw в разделе Presets

✅ Пресеты для Lightroom и Camera Raw — одинаковые .xmp файлы. Устанавливаются одни и те же файлы в разные папки.

LUT Для видео (.cube) — DaVinci / Premiere / Final Cut

DaVinci Resolve: Project Settings → Color Management → LUTs → Open LUT Folder → вставь .cube файлы → нажми Update Lists

Premiere Pro: Effects → Lumetri Color → Basic Correction → Input LUT → Browse → выбери .cube

Final Cut Pro: Скопируй в ~/Library/Application Support/Motion/Library/Looks → примени через Effects → Color → Custom LUT

💡 LUT'ы работают с любым материалом: log, Rec.709, клипы со смартфона. При необходимости скорректируй интенсивность (opacity).
