# 📸 avancato.ru

Персональный сайт-портфолио фотографа и веб-дизайнера **Антона Аванзато**.

> 📍 Казань | Профессиональная портретная фотография, веб-дизайн, обучение.

## 🛠 Стек технологий

*   **Платформа:** Wfolio (использование HTML-блоков)
*   **Разметка:** HTML5
*   **Стили:** CSS3 (Custom properties, Grid, Flexbox, Animations)
*   **Логика:** Vanilla JavaScript (без библиотек)
